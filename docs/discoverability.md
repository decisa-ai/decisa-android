# GitHub discoverability — decisa-android

## About (repository description)

```
Native Android attribution SDK (Kotlin / AAR) for Decisa — paid ads, web2app & funnel2app install matching via Play Install Referrer (dcs_mclid), deferred deep linking, and in-app conversion tracking. No GAID in v1.
```

Shorter alternative:

```
Android attribution SDK (Kotlin) for Decisa — deterministic web2app & funnel2app via Play Install Referrer, conversion tracking. Gradle AAR.
```
